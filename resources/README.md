# Resources Directory

## App Icon

To create a proper Windows app icon:

1. Create or design a 256x256 PNG icon for your application
2. Convert it to `.ico` format using an online converter or tool like:
   - https://convertio.co/png-ico/
   - https://icoconvert.com/

3. Save the `.ico` file as `icon.ico` in this directory

The icon should represent image compression (e.g., stacked images, compression symbol, or a simple graphic)

For now, the app will build without a custom icon and use a default Electron icon.
